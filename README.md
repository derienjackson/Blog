# Blog
<h2>Description</h2>
Blog creation app that utilizes CRUD operations to interact with blog posts stored using JSON server
<br />


<h2>Languages, Utilities, and Packages Used</h2>

- <b>React Native</b> 
- <b>Javascript<b/>
- JSON Server
- Ngrok
